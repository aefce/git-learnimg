git--version'檢查git版本'
git config --global user.name "your name" '設定使用者姓名'
git config --global user.email "your email"'設定使用者信箱'
git init'建立git歷史資料處存庫'
git status'檢查目前檔案狀況'
### 檔案狀態
#### Untracked-->尚未進入於git中
####    使用 "git add +檔名"將檔案放入Staged  
####    使用 "git add ."將所有變更檔案放入Staged  
#### Staged-->暫存區
####    使用 'git commit -m"填入訊息"'將檔案放入  Staged  
#### Commited-->提交
#### Tracked-->進入於git
